# Scripts

This directory contains automation scripts for the NTN prototype project.

## Available Scripts

Currently, test automation scripts are located in `/tests/on_target/`:
- `ntn_test.py` - Automated NTN connectivity testing
- `ntn_modem_commands.py` - AT command interface for modem diagnostics

## Future Scripts

This directory is reserved for:
- Build automation scripts
- Deployment automation
- CI/CD integration scripts
- Data collection and analysis tools
